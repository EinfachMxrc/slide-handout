Slide Handout

PowerPoint Add-in installieren

Der Add-in sendet die aktuelle Folien-Nummer aus PowerPoint an deine laufende Handout-Sitzung. So schalten sich an Folien gebundene Blöcke automatisch frei, während du sprichst.

manifest.xml herunterladen

Installation — PowerPoint Desktop (Windows & macOS)

  1. Einmaliges Setup: Netzwerkfreigabe oder lokaler Ordner als Trusted Catalog.
    • Windows: Einen Ordner erstellen (z. B. C:\OfficeAddins). Rechtsklick → Eigenschaften → Freigabe → als Netzwerkpfad freigeben.
    • macOS: In PowerPoint → Einstellungen → Sicherheit & Datenschutz → My Add-ins → Pfad auf einen iCloud/Desktop-Ordner setzen.
  2. PowerPoint → Datei → Optionen → Trust Center → Trust-Center-Einstellungen → Trusted Add-in Catalogs. Den Netzwerkpfad eintragen, Haken bei Im Menü anzeigen, PowerPoint neu starten.
  3. Die heruntergeladene slide-handout-addin.xml in den freigegebenen Ordner legen.
  4. PowerPoint → Einfügen → Meine Add-Ins → Shared Folder → Slide Handout wählen.
  5. Im Slide-Handout-Dashboard eine Sitzung starten → du siehst den 6-stelligen Pairing-Code. Im Add-in-Taskpane eingeben.

Installation — PowerPoint on the web (Office 365)

  1. PowerPoint im Browser öffnen → Einfügen → Add-ins → Mein Add-in hochladen.
  2. Die heruntergeladene slide-handout-addin.xml auswählen und bestätigen.
  3. Pairing-Code eingeben (siehe oben).

Fehlersuche

  • „Ungültiger Code" — nur Ziffern, genau 6 Stück.
  • „Nicht gefunden / abgelaufen" — die Sitzung wurde im Dashboard beendet oder es läuft gerade eine neue. Neu starten und den aktuellen Code nutzen.
  • „Zu viele Versuche" — 30 Requests/Minute Rate-Limit pro Code. Eine Minute warten.

Datenschutz: Der Add-in überträgt nur die aktuelle Folien-Nummer (1, 2, 3, …) — keine Folieninhalte, keine Bilder, keine Notizen.
